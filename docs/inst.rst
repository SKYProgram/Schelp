基本操作
==================================================

登录


   通过访问cloud.skyprogram.info:8000，可以进行登陆。
   
   用户会被重定向到登录页面，输入账号（电子邮件）以及密码即可进行登录。
   
   如果选择了“7天内保持登录”，我们将以cookie的形式标记一台计算机为：
   
      Logged <username>
